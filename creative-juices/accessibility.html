🌙 Dark Mode

Creative Juices uses a dark theme throughout and follows your device's system appearance setting.

iPhone / iPad Settings → Display & Brightness → Light / Dark
Android Settings → Display → Dark Theme

🔤 Text Size

The app automatically respects your device's text size settings.

iPhone / iPad Settings → Display & Brightness → Text Size
Android Settings → Display → Font Size

👁 Screen Reader

Creative Juices is built to work with your device's built-in screen reader.

iPhone / iPad — VoiceOver Settings → Accessibility → VoiceOver
Android — TalkBack Settings → Accessibility → TalkBack
